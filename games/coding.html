Concepts You Need to Know:

Let’s go over the basic concepts you’ll need for the challenges. Understanding these will help you fill in the blanks correctly!

1. Variables

A variable is a way to store data. In most programming languages, you assign a value to a variable using the = symbol.

For example, in Python:

name = "John" means the variable name contains the value "John".

In some languages like C#, you must first specify the type of data (e.g., int for numbers, string for text).

For example, in C#: int age = 25; declares a variable called age that holds the number 25.

2. Conditional Statements (if-else)

Conditional statements allow you to check if something is true or false and act accordingly.

For example, in JavaScript:

if (score > 90) { grade = "A"; } else { grade = "B"; }

This checks if the score is greater than 90. If it is, the grade is set to "A", otherwise, it is set to "B".

3. Loops

A loop allows you to repeat a set of actions. The most common loop is the for loop.

For example, in C#:

for (int i = 0; i < 5; i++) { Console.WriteLine(i); }

This means, "Start at 0, increase i by 1 each time, and stop when i is no longer less than 5."

The loop will print the numbers 0 to 4.

4. Functions

A function is a block of code that performs a specific task. You can call a function to run the code inside it.

For example, in Python:

def greet(name): print("Hello, " + name + "!")

This defines a function called greet that takes one input name and prints a greeting. You would call the function like this: greet("John").

Make sure you understand these concepts before moving to the next page!
