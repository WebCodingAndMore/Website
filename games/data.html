Concepts You Need to Know:

Let’s go over the basic concepts of data. Understanding these will help you fill in the blanks correctly!

1. Binary Numbers

Computers represent data using **binary**, a number system with only two digits: 0 and 1. These digits are called "bits."

Each bit in a binary number represents a different place value, similar to how each digit in a decimal number has a different place value (like 1s, 10s, 100s).

Here’s a breakdown of the binary number 01001010:

1286432168421
01001010

To convert the binary number 01001010 to decimal, add up the values of the places where there is a 1. In this case, it’s 64 + 8 + 2 = 74, so 01001010 = 74.

2. Adding Binary Numbers

Binary addition works similarly to decimal addition, but instead of carrying when you reach 10, you carry when you reach 2. Let’s add two binary numbers:

1011 (which is 11 in decimal) + 0100 (which is 6 in decimal).

Here’s what that looks like step by step:

Carry11100
1st Number1011
2nd Number0110
Sum10001

So, 1011 + 0110 = 10001, which is 17 in decimal.

3. ASCII Characters

**ASCII (American Standard Code for Information Interchange)** is a way to represent characters using numbers. Each character has a unique number assigned to it. For example:

  • A = 65
  • B = 66
  • a = 97
  • b = 98
  • space = 32

To convert from a character to its binary or decimal representation, you can look up its ASCII code.
