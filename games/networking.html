Concepts You Need to Know:

Let’s go over the basic concepts of networking. Understanding these will help you fill in the blanks correctly!

1. IP Addresses

Every device on a network has an **IP address**. This is a unique address that identifies devices on a network (like your phone, computer, or server).

There are two versions of IP addresses: **IPv4** and **IPv6**. IPv4 addresses are made up of four numbers between 0 and 255, separated by dots.

For example: 192.168.1.1 is a common IPv4 address.

2. TCP/IP

TCP/IP is the **language** of the internet. It is the protocol that defines how data is broken down into smaller pieces called **packets** and sent across networks. The receiving device then **reassembles** the packets to read the data.

TCP ensures that packets are delivered in the correct order and without errors.

3. HTTP

HTTP (HyperText Transfer Protocol) is the foundation of the **World Wide Web**. It is how browsers communicate with servers to request and display web pages.

When you type a URL like https://example.com in your browser, it sends an **HTTP request** to the server asking for the web page, and the server responds with the page data.

4. DNS (Domain Name System)

DNS is like the **phonebook** of the internet. It translates **domain names** (like www.example.com) into IP addresses (like 192.168.1.1) so that computers can communicate with each other.

When you enter a website’s name into the browser, a DNS server looks up the IP address for that domain name and helps your device connect to the correct server.
