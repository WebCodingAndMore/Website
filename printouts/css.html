Concepts You Need to Know:

Let’s go over the basic concepts of CSS. Understanding these will help you fill in the blanks correctly!

1. CSS Selectors

CSS selectors are used to select HTML elements you want to style. You can select elements by their tag name, class, or ID.

For example:

  • Tag selector: h1 selects all <h1> elements.
  • Class selector: .class-name selects all elements with the class class-name.
  • ID selector: #id-name selects the element with the ID id-name.

For example, h1 { color: red; } would make all <h1> elements red.

2. CSS Properties and Values

CSS styles elements using properties and values. A property controls an aspect of the element’s style, and the value defines what that style should be.

For example, to set the text color to red, you use the color property and assign the value red:

h1 { color: red; }

Some common properties include:

  • color: Changes the text color.
  • background-color: Changes the background color.
  • font-size: Sets the size of the text.

3. CSS Colors

Colors can be set using names like red, blue, or green, or by using HEX codes (e.g., #ff0000 for red) or RGB values (e.g., rgb(255, 0, 0) for red).

For example, color: #00ff00; would make the text green.

4. The Box Model

In CSS, every element is a "box." The box model consists of four areas:

  • Content: The actual content of the box (like text or an image).
  • Padding: Space between the content and the border.
  • Border: The line around the padding (or content if there’s no padding).
  • Margin: Space outside the border, between this element and others.

For example:

div { padding: 10px; border: 1px solid black; margin: 20px; }

Make sure you understand these concepts before moving to the next page!
