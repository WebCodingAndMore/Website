Concepts You Need to Know:

Let’s go over the basic concepts of HTML. Understanding these will help you fill in the blanks correctly!

1. HTML Tags

HTML is made up of tags that tell the browser what each part of a webpage should do.

An HTML tag looks like this: <tagname>content</tagname>. The opening tag <tagname> starts the element, and the closing tag </tagname> ends the element.

For example, to create a heading, you use <h1> for a large heading:

<h1>Welcome to My Website!</h1>

The tag <h1> is an opening tag, and </h1> is a closing tag. Everything between these tags will be displayed as a large heading.

2. Basic HTML Structure

All HTML documents have the same basic structure. It starts with the <!DOCTYPE html> declaration, followed by the <html> tag, which contains the entire webpage.

Inside the <html> tag, you will find two sections: <head> and <body>.

  • The <head> section contains information about the webpage, such as its title.
  • The <body> section contains the content of the webpage that will be displayed to the user.

For example:

<html>

   <head>

      <title>My Webpage</title>

   </head>

   <body>

      <h1>Hello, World!</h1>

   </body>

</html>

3. Links

Links allow users to navigate to different web pages. You create a link using the <a> tag. The href attribute tells the browser where the link should go.

For example, to link to Google, you would write:

<a href="https://www.google.com">Go to Google</a>

The text Go to Google will appear as a clickable link.

4. Lists

There are two types of lists in HTML:

  • Ordered lists (<ol>) which are numbered
  • Unordered lists (<ul>) which use bullet points

Each item in a list is created with the <li> tag. For example:

<ul>

   <li>Apple</li>

   <li>Banana</li>

   <li>Cherry</li>

</ul>

Make sure you understand these concepts before moving to the next page!
