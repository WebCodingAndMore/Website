Concepts You Need to Know:

Let’s go over the basic concepts of JavaScript. Understanding these will help you fill in the blanks correctly!

1. Variables

Variables store data that you can use later in your program. In JavaScript, you can declare variables using the keywords let, const, or var.

For example, to store a number, you would write:

let age = 25; means the variable age stores the value 25.

2. Conditional Statements (if-else)

Conditional statements allow you to execute certain code only if a condition is true.

For example:

if (age > 18) { console.log("You are an adult."); }

This checks if age is greater than 18. If it is, the message "You are an adult." is printed to the console.

3. Loops

Loops allow you to repeat a block of code a certain number of times. The most common loop is the for loop.

For example, to print the numbers 0 through 4, you would write:

for (let i = 0; i < 5; i++) { console.log(i); }

This loop starts at 0, and each time it runs, it increases i by 1. The loop stops when i is no longer less than 5.

4. Functions

Functions are blocks of code that you can call to perform a specific task. You define a function using the keyword function, followed by the function name and parentheses.

For example, you can define a function to greet someone by name:

function greet(name) { console.log("Hello, " + name + "!"); }

To call this function and greet someone, you would write: greet("John");
